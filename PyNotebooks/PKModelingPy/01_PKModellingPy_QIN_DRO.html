Reading DCE-MRI image meta-data¶

The DCE-MRI DICOM to NRRD conversion is done in 3D Slicer by reading the DICOM files with Multivolume image plugin [reference] and saving the image volume node as nrrd.
The parameters we need for PK Modellin are: * Sampling Rate * Flip Angle (FA) * Repetition Time (TR)

PK Modelling¶

In [10]: 
bc = BATCalculator()
# Solve PK modelling
solver = PKSolver()
ndaKtrans = np.zeros((m,n,p))
ndaVe = np.zeros((m,n,p))
ndaRsquared = np.zeros((m,n,p))
ndaBATCt = np.zeros((m,n,p))
ndaCtConcentration = np.zeros((m,n,p,q))
for i in xrange(0,m):
    for j in xrange(0,n):
        for k in xrange(0,p):
            if ndaLabel[i,j,k] == 1:
                st = ndaImage[i,j,k,:] # Tissue Signal
                # Tissue signal to concentration conversion
                converter.signal = st
                converter.FA = FA
                converter.TR = TR
                converter.T10 = 1000
                converter.BAT = 10
                converter.Relaxivity = 0.0045
                TRF = converter.convert()
                # Assigning the concentration curve
                ndaCtConcentration[i,j,k,:] = TRF
                solver.AIF = aif
                solver.TRF = TRF
                solver.BATCp = 15 
                solver.BATCt = 16
                ndaBATCt[i,k,k] = solver.BATCt
                solver.samplingRate = samplingRate
                solver.hematocrit = 0.45
                result = solver.solve()
                ndaKtrans[i,j,k] = result[0]
                ndaVe[i,j,k] = result[1]
                ndaRsquared[i,j,k] = solver.calculateGoodnessOfFit()
